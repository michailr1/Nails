# WEB-BOOKING-EDIT

## Цель

Сделать календарь рабочего кабинета мастера изменяемым: будущую активную запись можно перенести или отменить без обращения к Telegram-диалогу.

## Инварианты

- история записи не удаляется;
- отмена меняет статус, а не удаляет строку;
- перенос использует существующие серверные правила доступности и конфликтов;
- все операции owner-scoped;
- мутации защищены действующей web-сессией и CSRF-механизмом;
- завершённые и уже отменённые записи нельзя менять;
- после мутации календарь перечитывается с сервера;
- никаких автоматических сообщений клиентке.

## UX

1. Нажатие на запись открывает её карточку.
2. «Перенести» показывает дату и время с текущими значениями.
3. «Отменить запись» требует отдельного подтверждения.
4. Серверные ошибки объясняются человеческим языком.
5. На мобильном действия занимают полную доступную ширину.
