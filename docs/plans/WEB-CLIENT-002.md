# WEB-CLIENT-002 — добавление и архивирование клиенток

## Цель

Сделать раздел «Клиентки» рабочим: мастер может добавить новую карточку и отправить выбранную карточку в архив.

## Правила

- создание доступно прямо из раздела «Клиентки»;
- достаточно имени, телефон необязателен;
- форма открывается заметно в верхней части страницы;
- используется существующий `create_or_reuse_client`;
- архивирование требует подтверждения с именем клиентки;
- архивирование меняет существующий `ClientProfileStatus` на `archived`;
- история записей не удаляется;
- архивная карточка исчезает из активного списка и из «Давно не были»;
- физического удаления и нового статуса нет.
