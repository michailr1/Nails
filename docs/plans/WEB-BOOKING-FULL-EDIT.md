# WEB-BOOKING-FULL-EDIT — полное редактирование записи и рабочая цена

## Цель

Запись должна обслуживаться Нэйли, а не требовать обязательного ручного уточнения мастером.

## Правила цены

- фиксированные цены складываются автоматически;
- диапазоны учитываются по нижней границе и показываются с «от»;
- цена за единицу в первой версии учитывается как одна единица;
- известная часть смешанного состава не теряется;
- ручная итоговая цена имеет приоритет только после осознанной корректировки мастера;
- прошедшая запись входит в статистику по сохранённой рабочей сумме без обязательной финализации.

## Редактирование записи

В кабинете мастер может изменить:

- клиентку;
- основную процедуру;
- дополнения;
- дату и время;
- итоговую цену;
- длительность.

Сервер заново разрешает owner-scoped карточку и активные позиции прайса, пересчитывает snapshots, резерв и рабочую цену, проверяет выходной и пересечения с исключением самой редактируемой записи. Перед сохранением интерфейс показывает полную сводку и требует подтверждение.

## Инварианты

- мутация owner-scoped;
- история не удаляется;
- audit не содержит private-полей;
- клиентке ничего автоматически не отправляется;
- ручная цена и длительность остаются только overrides конкретной записи;
- после сохранения календарь перечитывается с сервера.
